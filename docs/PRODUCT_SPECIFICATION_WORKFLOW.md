# Product Specification Publication

## Source

The supplied SS Bricks catalogue is the approved publication source for the four
website product lines. Website values and downloadable specification sheets must
remain aligned with this source.

## Published Catalogue Values

| Product | Dimensions | Unit weight | Compressive strength | Water absorption |
| --- | --- | --- | --- | --- |
| Solid Cement Blocks | 12 x 8-9 x 6 inches | 18-20 kg | 10 N/mm2 | 6-9% |
| Fly Ash Bricks | 9 x 4.5 x 3 inches | 2.2-2.8 kg | 15-17.5 N/mm2 | 12-15% |
| Paver Blocks | 9 x 4.5 x 3 inches | 2.2-2.5 kg | 16-18 N/mm2 | 10-12% |
| Mud Bricks | 9 x 4.5 x 3 inches | 2.8-3 kg | 10-12 N/mm2 | 9-12% |

The catalogue also presents a high-strength Solid Cement Block variant at
15 N/mm2, 5-8% water absorption, and 20-22 kg per piece. That variant is not
combined with the standard block values listed above.

## Publication Rules

1. Use the catalogue values above on the public product page.
2. Generate all four product specification PDFs from the same values.
3. Do not show packaging, pallet, or truckload quantities on the product page or
   specification sheets.
4. Keep the high-strength Solid Cement Block variant separate from the standard
   product if it is added in the future.
5. Recheck the website and all PDFs whenever the source catalogue changes.
